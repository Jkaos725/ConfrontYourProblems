## How to Run the Code

The base code was made in **Godot**.

You can run and view the project on desktop from here:  
https://godotengine.org/

Make sure to:

- Run **Godot v4.6.1-stable** without the console
- Clone the project
- Open Godot
- Import the project
- Go to the project location
- Click the project file
- Open it in the editor

## API Key Setup

In the local project folder, create a file named `Config.cfg`.

Add a Groq key like this:

```ini
[application]
groq_api_key="API-KEY"
```

## Run Without Godot

You can also download the different release builds and run them directly on the computer.

## Project Idea

The theme was **AI Solutions to College Level Problems**.

We were stuck on the theme for a bit, since we were mostly thinking of chatbots and similar ideas.

Then we came up with the idea of making an escape room with an AI tutor.

Jacob decided to use Godot since he was already used to it, and the scripting language has syntax similar to Python, which made it easier to work with.

We started from there and kept building on the idea as we went.

The idea grew as we added new features and talked with other people about it.

We decided to go with an escape room powered by an AI tutor and build from that concept.

Godot was chosen because it was familiar for development, and GDScript has syntax similar to Python.
